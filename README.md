## 📌 Overview

An end-to-end **Generative AI system** that transforms structured
investment data into:

✨ Investor Biographies\
✨ Investor Theses\
✨ Company Profiles\
✨ Funding-Based Blogs

------------------------------------------------------------------------

## 🧠 Architecture

``` mermaid
graph TD
A[Azure Cosmos DB] --> B[Data Processing]
B --> C[Rule-Based Templates]
C --> D[Prompt Engineering]
D --> E[LLM - Mistral 7B]
E --> F[Final Output]
```
